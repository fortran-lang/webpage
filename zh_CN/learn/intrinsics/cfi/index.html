c_associated#

名称#

c_associated(3) - [ISO_C_BINDING] C 指针的状态

语法#

result = c_associated(c_prt_1, c_ptr_2)

说明#

c_associated(c_prt_1[, c_ptr_2]) 确定 C 指针 c_ptr_1 的状态或 c_ptr_1 是否与目标 c_ptr_2 相关联。

参数#

  • c_ptr_1

    c_ptr 或 c_funptr 类型的标量。

  • c_ptr_2

    （可选）与 c_ptr_1 类型相同的标量。

返回#

返回值的类型为 logical; 如果 c_ptr_1 是 C NULL 指针或者 c_ptr1 和 c_ptr_2 指向不同的地址则返回 .false.。

示例#

示例程序：

program demo_c_associated

contains

subroutine association_test(a,b)
use iso_c_binding, only: c_associated, c_loc, c_ptr
implicit none
real, pointer :: a
type(c_ptr) :: b
   if(c_associated(b, c_loc(a))) &
      stop 'b and a do not point to same target'
end subroutine association_test

end program demo_c_associated

标准#

Fortran 2003 及以后

另见#

c_loc(3)，c_funloc(3)，iso_c_binding(3)

c_f_pointer#

名称#

c_f_pointer(3) - [ISO_C_BINDING] 将 C 指针转换为 Fortran 指针

语法#

call c_f_pointer(cptr, fptr, shape)

说明#

c_f_pointer(cptr, fptr[, shape]) 将目标C 指针 cptr 分配给 Fortran 指针 fptr 并指定其形状。

参数#

  • cptr

    c_ptr 类型的标量。它是 intent(in) 的。

  • fptr

    可与 cptr 互操作的指针。它是 intent(out) 的。

  • shape

    （可选）具有 intent(in) 类型的 integer 一维数组。当且仅当 fptr 是一个数组时，它才会出现。大小必须等于 fptr 的维度。

示例#

示例程序：

program demo_c_f_pointer
use iso_c_binding
implicit none
interface
   subroutine my_routine(p) bind(c,name='myC_func')
      import :: c_ptr
      type(c_ptr), intent(out) :: p
   end subroutine
end interface
type(c_ptr) :: cptr
real,pointer :: a(:)
   call my_routine(cptr)
   call c_f_pointer(cptr, a, [12])
end program demo_c_f_pointer

标准#

Fortran 2003 及以后

另见#

c_loc(3)，c_f_procpointer(3)，iso_c_binding(3)

c_f_procpointer#

名称#

c_f_procpointer(3) - [ISO_C_BINDING] 将 C 函数指针转换为 Fortran 过程指针

语法#

call c_f_procpointer(cptr, fptr)

说明#

c_f_procpointer(cptr, fptr) 将 C 函数指针 cptr 的目标分配给 Fortran 过程指针 fptr。

参数#

  • cptr

    c_funptr 类型的标量。它是**intent(in)**的。

  • fptr

    可与 cptr 互操作的过程指针。它是 intent(out)。

示例#

示例程序：

program demo_c_f_procpointer
use iso_c_binding
implicit none
abstract interface
   function func(a)
   import :: c_float
   real(c_float), intent(in) :: a
   real(c_float) :: func
   end function
end interface
interface
   function getIterFunc() bind(c,name="getIterFunc")
   import :: c_funptr
   type(c_funptr) :: getIterFunc
   end function
end interface
type(c_funptr) :: cfunptr
procedure(func), pointer :: myFunc
   cfunptr = getIterFunc()
   call c_f_procpointer(cfunptr, myFunc)
end program demo_c_f_procpointer

另见#

c_funloc#

名称#

c_funloc(3) - [ISO_C_BINDING] 获取一个过程的C地址

语法#

result = c_funloc(x)

说明#

c_funloc(x) 确定参数的 C 地址。

参数#

  • x

    可互操作的函数或指向此类函数的指针。

返回#

返回值是 c_funptr 类型，包含参数的 C 地址。

示例#

示例程序：

! program demo_c_funloc and module
module x
use iso_c_binding
implicit none
contains
subroutine sub(a) bind(c)
real(c_float) :: a
   a = sqrt(a)+5.0
end subroutine sub
end module x
!
program demo_c_funloc
use iso_c_binding
use x
implicit none
interface
   subroutine my_routine(p) bind(c,name='myC_func')
     import :: c_funptr
     type(c_funptr), intent(in) :: p
   end subroutine
end interface
   call my_routine(c_funloc(sub))
!
end program demo_c_funloc

c_loc#

名称#

c_loc(3) - [ISO_C_BINDING] 获取对象的C地址

语法#

result = c_loc(x)

说明#

c_loc(x) 确定参数的 C 地址。

参数#

  • x

    Shall have either the pointer or target attribute. It shall not be a coindexed object. It shall either be a variable with interoperable type and kind type parameters, or be a scalar, nonpolymorphic variable with no length type parameters.

返回#

返回值是 c_ptr 类型，包含参数的 C 地址。

示例#

示例程序：

   subroutine association_test(a,b)
   use iso_c_binding, only: c_associated, c_loc, c_ptr
   implicit none
   real, pointer :: a
   type(c_ptr) :: b
     if(c_associated(b, c_loc(a))) &
        stop 'b and a do not point to same target'
   end subroutine association_test

c_sizeof#

名称#

c_sizeof(3) - [ISO_C_BINDING] 表达式的字节大小

语法#

n = c_sizeof(x)

说明#

c_sizeof(x) 计算表达式 x 占用的存储字节数。

参数#

  • x

    参数应该是一个可互操作的数据实体。

返回#

返回值是整数类型和系统相关类型 csize_t（来自 _iso_c_binding 模块）。它的值是参数占用的字节数。如果参数有 pointer 属性，则返回指向的存储区域的字节数。如果参数是具有 pointer 或 allocatable 成员的派生类型，则返回值不考虑这些成员指向的数据的大小。

示例#

示例程序：

program demo_c_sizeof
use iso_c_binding
implicit none
real(c_float) :: r, s(5)
   print *, (c_sizeof(s)/c_sizeof(r) == 5)
end program demo_c_sizeof

结果：

    T

该示例将打印 .true。除非你使用的平台默认 real 变量被异常填充。
